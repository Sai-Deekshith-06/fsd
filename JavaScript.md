# JavaScript

- It is a scripting language used in web-pages to make them dynamic in nature.
- It can be used on both client-side and sever-side.
- Browsers contain a JS engines like V8Engine, SpiderMonkey, chakra to interpret JavaScript.
- To make interactive and dynamic web-pages, validate fields, data-processing. connecting to backend.
- Loosely-typed (variable declaration and data-types)
