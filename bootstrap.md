# Bootstrap

- SASS(Syntactically Awesome Style Sheets)
- Containers: 
1. Container => fixed width;width chamges based on screen size
2. Container-fluid => Occupies the width of viewport(100vw); Full width
3. Cntainer-{breakpoint} => 100% wide until the specified breakpoint is reached

- Screen sizes: xs(default), sm, md, lg, xl, xxl
    - t => top
    - b => bottom
    - s => start/left
    - e => end/right
    - x => left and right
    - y => top and bottom

- Typography
    - `lead` to highlight
    - `text-uppercase` to uppercase
    - `text-lowercase` to lowercase
